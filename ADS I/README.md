# ADS - 1º Semestre de 2026

O objetivo deste repositório é armazenar e organizar os materiais produzidos durante todos os semestres do curso de Análise e Desenvolvimento de Sistemas, facilitando o acesso aos documentos acadêmicos e mantendo um histórico das atividades realizadas.

## Engenharia de Software I

Nesta matéria foram desenvolvidos documentos relacionados à análise, levantamento de requisitos, estudo de caso, atores do sistema e casos de uso.  
O foco principal foi compreender melhor o funcionamento de sistemas, identificar necessidades dos usuários e organizar as funcionalidades antes da etapa de desenvolvimento.

### Arquivos

- `(ENGENHARIA DE SOFTWARE I) - SISTEMA DE FORMULÁRIO WEB P1.docx`
- `ATORES FUNCIONAIS.xlsx`
- `Caso de Uso - Aplicativo de combate a Dengue.docx`
- `Estudo de caso - projeto ADS.docx`

### O que foi feito

- Criação de estudo de caso para projeto acadêmico.
- Levantamento e organização de atores funcionais.
- Desenvolvimento de caso de uso para aplicativo de combate à dengue.
- Estruturação de um sistema de formulário web.
- Organização de informações voltadas à modelagem e documentação de software.

## Projeto Integrador

Nesta matéria foi desenvolvida uma análise comparativa sobre formulários de pesquisa, com o objetivo de estudar ferramentas existentes e entender suas funcionalidades, vantagens e limitações.

### Arquivo

- `(PROJETO INTEGRADOR) - ANÁLISE COMPARATIVA DE FORMULÁRIOS DE PESQUISA - P1 - ADS 1.docx`

### O que foi feito

- Análise comparativa de plataformas de formulários de pesquisa.
- Estudo das funcionalidades oferecidas por diferentes ferramentas.
- Comparação entre recursos, usabilidade e possibilidades de aplicação.
- Organização das informações para apoiar o desenvolvimento do projeto acadêmico.
